Установка.

Скачайте python на python.org и установите.
При установке выберите "для всех пользователей" и "прописать путь"
Скопируйте библиотеку metahash.py в папку lib куда встал python.
Для удобстав использования также рекомендуется установить Far manager.
Папку со скриптами можно положить в любое произвольное место.

Запуск скриптов.

Начните с проверки баланса.
В Far manager откройте balance.py по F4.
Впишите туда свой адрес. Или вообще любой баланс которого вы хотите узнать.
Для запуска в Far просто нажмите Enter на файле balance.py.

При первом запуске вам выдаст ошибку что не установлены модули.
Их надо установить набрав команды:
pip install cryptography
pip install requests
Можно еще обновить pip (будет желтым написано сообщение).

Для остальных операций вам так же потребуется в скрипте в верхнем блоке
исправить адрес а так же положить ваш приватный ключ. Без ключа отправка
делегирование и отправка работать не будут и скрипт выдвст сообщение
об ошибке.
